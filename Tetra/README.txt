# Tetris Game in Python with Pygame

A classic Tetris game built using Python and the Pygame library, featuring customizable graphics, shape flipping, and dynamic gameplay.

## Features

- **Colorful Background**: A gradient background that changes dynamically during gameplay.
- **Shape Flipping**: Flip Tetromino shapes vertically with the `Up Arrow` key for added flexibility.
- **Score Tracking**: Tracks and displays the player's score in real-time as rows are cleared.
- **Customizable**: Easily modify game constants, such as grid size, block colors, and speed.

## Gameplay Controls

- **Left Arrow**: Move the Tetromino left.
- **Right Arrow**: Move the Tetromino right.
- **Down Arrow**: Speed up the Tetromino's descent.
- **Up Arrow**: Flip the current Tetromino vertically.
